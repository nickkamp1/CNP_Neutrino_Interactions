# Cached data files

Generated by `../make_cache.py` on a full-toolchain machine, then either committed here (so a Colab
`git clone` brings them) or uploaded to your `CACHE_BASE_URL`. Expected files:

| File | Built by | `make_cache.py` | Used in |
|---|---|---|---|
| `atmo_flux_siren.npz` | SIREN flux tables (Zenodo 20129082) | `--flux` | 3a flux, surface-vs-detector, weights |
| `siren_nutau_injection.parquet` | SIREN injection | `--siren` | 3a interaction, weights |
| `tau_decays_proposal.parquet` | PROPOSAL tau propagation | `--tau` | 3b decay length / double-bang |
| `prometheus_nutau_example.parquet` | Prometheus | `--events` | 3c event display |

These are **real** outputs. If a file is missing, the corresponding loader raises a clear error naming
the command to run — the notebooks never substitute synthetic data.
