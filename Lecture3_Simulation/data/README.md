# Cached data files

Generated by `../make_cache.py` on a full-toolchain machine, then uploaded to your
`CACHE_BASE_URL`. The notebook downloads them here on Colab. Expected files:

| File | Built by | Used in |
|---|---|---|
| `atmo_flux_siren.npz` | SIREN flux tables (Zenodo 20129082) | §1 flux, §1b surface-vs-detector, §6 weights |
| `siren_nutau_injection.parquet` | SIREN | §2 interaction, §6 weights |
| `prometheus_nutau_example.parquet` | Prometheus | §5 event display |

If a file is missing the notebook falls back to a clearly-labelled synthetic placeholder so the
lecture still runs.
